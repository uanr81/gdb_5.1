target adapt устр

Монитор Adapt для A29K.

target amd-eb устр скорость прог

Удаленная PC-резидентная плата AMD EB29K, присоединенная по последовательным линиям. Устр является последовательным устройством, так же, как для target remote; скорость позволяет вам указать скорость линии; а прог является именем программы, которая будет отлаживаться, так, как оно появляется в ДОС на ПК. See Протокол EBMON для AMD29K.
