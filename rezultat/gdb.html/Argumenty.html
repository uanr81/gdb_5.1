4.3 Аргументы вашей программы

Аргументы к вашей программе могут быть заданы как аргументы к команде run. Они передаются оболочке, которая раскрывает символы шаблонов и выполняет перенаправление ввода-вывода, и с того момента попадают в вашу программу. Переменная среды SHELL (если она существует) определяет, какую оболочку использует GDB. Если вы не определите SHELL, он использует оболочку по умолчанию (/bin/sh в Unix).

В не-Unix-системах, программу обычно запускает непосредственно GDB, который эмулирует перенаправление ввода-вывода через соответствующие системные вызовы, и символы шаблонов раскрываются кодом запуска, а не оболочкой.

run без аргументов использует те же аргументы, которые использовались предыдущей командой run, или которые установлены командой set args.

set args

Задает аргументы, которые будут использоваться при следующем запуске вашей программы. Если у set args нет аргументов, run выполняет вашу программу без аргументов. Если вы запустили программу с аргументами, то единственный способ запустить ее снова без аргументов — это использовать set args до следующего запуска командой run.

show args

Показать аргументы, которые будут переданы вашей программе при ее вызове.
